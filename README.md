# Luis Humberto Lopez — Foreign Legal Consultant (Mexican Law)

Static bilingual website (English/Spanish) with 4 pages:
- Home
- Areas of Practice
- Background
- Contact Information

## Local preview
Open `index.html` directly, or run a local server:

```bash
python3 -m http.server 5500
```

Then visit:
http://localhost:5500

## Publish with GitHub Pages (free)
1. Upload all files to the repo root (same level as this README).
2. Go to **Settings → Pages**.
3. Source: **Deploy from a branch**
4. Branch: **main**, Folder: **/(root)**
5. Save, then use the URL GitHub provides.
